INTRODUCTION
------------

The Color-game is a front-end based web app that displays different a RGB(Red-Green-Blue) color value and shows the right color and other wrong colors.

The user is to pick the color that corresponds to the RGB value given.

Picking the right color allows the user to restart the game, a wrong color pick allows the user to pick again.

It uses jQuery for the manipulation of DOM elements.

CONFIGURATION
-------------

The module has no menu. There is no configuration.
